# Planet Name Mapping: Earth Cities → Infestation Planets

This document maps the original 48 Pandemic cities to new sci-fi planet names for the Infestation game.

## Galaxy Color Mapping
- **Blue** (North America/Europe) → **Orion Sector**
- **Yellow** (South America/Africa) → **Hydra Sector**
- **Black** (Asia/Middle East) → **Nebula Sector**
- **Red** (East Asia/Oceania) → **Phoenix Sector**

## Starting Planet
- **Original**: Atlanta
- **New**: Nova Haven (Orion Sector starting command base)
